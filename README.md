## 📚 Sobre

<p>
Criado pela <a href="https://rocketseat.com.br/" target="_blank">Rocketseat</a>, este é um curso livre para aprendizado da estilização de páginas web.

Lecionado por Mayk Brito, o curso "Alinhando os Planetas" mostra o alinhamentos de layouts utilizando Flexbox.

### Grade do curso

<details>
  <summary>Layouts CSS</summary>

- Layouts e evolução
- Normal flow
- Table
- Tableless
- Flexbox
</details>

<details>
  <summary>Terminologia</summary>

- Flex container - Flex item e Nesting
- Eixo principal e cruzado
- Flex sizing
</details>

<details>
  <summary>Propriedades do Flex Container</summary>

- Propriedades do Flex Container
- Direção dos itens
- Multiplas linhas
- Direção e multi linha em uma única propriedade
- Alinhamento dos itens no eixo principal
- Alinhamento dos itens no eixo cruzado
- Espaço entre os elementos
</details>

<details>
  <summary>Propriedades para os itens</summary>

- Propriedades para os itens
- Largura e altura dos itens com flex-basis
- Crescimento e adaptação dos itens
- Encolhimento e encaixe dos elementos
- Shorthand flex
- Alterando tamanho de múltiplos itens
- Ordenando itens
</details>

<details>
  <summary>Desafios</summary>

- Header menu navigation
- Layout com 2 colunas
- 9 fotos 3 colunas
- Imagem dentro de um botão
- Footer
</details>
